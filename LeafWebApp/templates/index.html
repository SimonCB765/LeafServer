There are two methods of culling provided by the server.

  1. Subsets of the PDB can be culled by pairwise sequence identity and structure quality. The subsets can be defined by the user uploading a set of PDB chains/entries, or by the user selecting to cull the proteome of a given organism.
  2. A user can upload a set of proteins in FASTA format, and a non-redundant dataset can be generated from the uploaded sequences.

Pre-computed sets of nonredundant PDB chains can also be downloaded, along with the source code and data files needed to run the culling on a local machine.
