Pre-culled Datasets

This page provides downloads for non-redundant datasets, generated using a variety of structural criteria.

If you want to generate a non-redundant dataset using different structural criteria, this can be done by generating a user requested dataset.

The name of each download follows a defined format. The format is: SeqIden_X_Res_Y_RVal_Z. X is the percentage sequence identity threshold used. This means that if X is 20, then there are no two sequences in the non-redundant dataset witha pairwise sequence idenitty greater than 20%. Y is the maximum resolution of the protein structure. Z is the maximum R value. If the name of the download ends in INCLNONXRAY_INCLCAONLY, then the dataset from which the redundancy was removed contained non-X-ray structures, and chains where the strucutre contains only backbone alpha carbon atoms.

The non-redundant dataset is recorded in a FASTA file, with the same format as the FASTA files returned by the user generated redundancy removal. The format of the files can be found here, ignoring the portion about the pdbaaent dataset, pdbaanr dataset and representative sequences.

FASTA Files of the Entire PDB

For a more in depth description of the format and process of generating the datasets, see here.

pdbaa.gz - Every protein chain in the PDB is included once in pdbaa. An example of the format of the description line is as follows:

>1GA2A  614  XRAY  1.7  0.17  0.197  no  ENDOGLUCANASE 9G    [Clostridium cellulolyticum]

pdbaaent.gz - Only those chains which are non-redundant withinn each entry (e.g. 1GA2) appear in pdbaaent. For example, 1GA2B has an identical sequence to 1GA2A, and therefore does not appear in pdbaaent. In this file, the description line for 1GA2A changes to:

>1GA2A  614  XRAY  1.7  0.17  0.197  no  ENDOGLUCANASE 9G    [Clostridium cellulolyticum] || 1GA2B

pdbaanr.gz - Only those chains which are non-redundant withinn the entire PDB are included in pdbaanr. For example, 1KFGB, 1KFGA, 1K72B, 1K72A and 1GA2B all have the same sequence as 1GA2A, and therefore do not appear in pdbaanr. In this file, the description line for 1GA2A changes to:

>1GA2A  614  XRAY  1.7  0.17  0.197  no  ENDOGLUCANASE 9G    [Clostridium cellulolyticum] || 1KFGB 1KFGA 1K72B 1K72A 1GA2B

Downloads for Local Redundancy Removal

The source code and data files needed to run Leaf locally. The PDB data files are only necessary if you want to cull the PDB locally. Two different versions of the source code are provided. Download the one that is appropriate for your local OS. For more information on how to run the program, and what data is needed, see the README included with the source code.

Windows Source Code - The source code and BLAST executables needed to run the redundancy removal on Windows machines.

Linux Source Code - The source code and BLAST executables needed to run the redundancy removal on Linux machines.

PDB Data Files - Up to date processed PDB data files. These are needed to run the PDB redundancy removal on a local machine.
